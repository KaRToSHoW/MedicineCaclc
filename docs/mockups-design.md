# Mockup экранов приложения "Медицинский калькулятор"

## Описание дизайнерского решения

### Дизайн-концепция
Приложение следует принципам **медицинского Material Design** с акцентом на:
- **Профессиональность**: чистый, минималистичный интерфейс
- **Читаемость**: крупные шрифты, высокий контраст
- **Доступность**: интуитивная навигация, ясная иерархия
- **Доверие**: спокойная цветовая палитра, медицинская тематика

### Цветовая схема

**Основные цвета:**
- Primary (Синий): `#0080FF` - основные действия, акценты
- Secondary (Фиолетовый): `#7B61FF` - дополнительные элементы
- Success (Зеленый): `#10B981` - положительные результаты
- Warning (Оранжевый): `#F59E0B` - предупреждения
- Danger (Красный): `#EF4444` - критичные значения

**Нейтральные цвета:**
- Surface: `#FFFFFF` - фон карточек
- Surface Secondary: `#F5F5F7` - фон экрана
- Text Primary: `#1F2937` - основной текст
- Text Secondary: `#6B7280` - вторичный текст
- Border: `#E5E7EB` - границы элементов

### Типографика
- **Заголовки**: System Bold, 24-32px
- **Подзаголовки**: System Semibold, 18-20px
- **Основной текст**: System Regular, 14-16px
- **Мелкий текст**: System Regular, 12-13px

## Дизайн-система: Компоненты

### Кнопки
**Primary Button** - основное действие
- Цвет: Primary blue `#0080FF`
- Текст: белый
- Border radius: 12px
- Padding: 14px 24px

**Secondary Button** - вторичное действие
- Цвет: прозрачный
- Текст: primary blue
- Border: 2px solid primary
- Border radius: 12px

**Danger Button** - удаление
- Цвет: Danger red `#EF4444`
- Текст: белый
- Border radius: 12px

### Карточки
**Card** - стандартная карточка
- Background: White `#FFFFFF`
- Border: 1px solid `#E5E7EB`
- Border radius: 16px
- Shadow: subtle drop shadow
- Padding: 16px-20px

**Interactive Card** - интерактивная карточка
- Hover: легкое поднятие (shadow увеличивается)
- Active: opacity 0.9

### Иконки
- Использование эмодзи для медицинских категорий
- Системные иконки для UI элементов
- Размеры: 20px, 24px, 32px, 48px

### Spacing
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px

### Анимации
- Transition duration: 200ms
- Easing: ease-in-out
- Используются для:
  - Навигация между экранами
  - Появление/исчезновение элементов
  - Интерактивные нажатия

## Адаптивность

Приложение адаптируется под:
- **Мобильные устройства**: 375px - 428px (iPhone)
- **Планшеты**: 768px - 1024px (iPad)
- **Web**: 1280px+ (Desktop browsers)

Ключевые принципы:
- Вертикальная компоновка на мобильных
- Двух-колоночный layout на планшетах
- Сайдбар + контент на desktop

## Accessibility (Доступность)

- **Контрастность**: WCAG AA соответствие
- **Размер текста**: минимум 14px
- **Touch targets**: минимум 44x44px
- **Screen readers**: семантические теги
- **Keyboard navigation**: для web версии
